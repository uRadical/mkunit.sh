Installation

Install mkunit on your system.

Requirements

  • Linux with systemd (for full functionality)
  • macOS (for development/testing - unit files can be generated but not installed)

Homebrew (Recommended)

The easiest way to install mkunit on Linux or macOS:

brew install mkunit

This automatically sets up shell completions for bash, zsh, and fish.

Cargo (Rust)

If you have Rust installed:

cargo install mkunit

Make sure ~/.cargo/bin is in your PATH:

# Add to ~/.bashrc or ~/.zshrc
export PATH="$HOME/.cargo/bin:$PATH"

Install Script (Debian/Ubuntu)

For Debian-based systems:

curl -sSL https://mkunit.sh/install.sh | bash

This downloads the latest release binary and installs it to /usr/local/bin.

Note

Review scripts before running them: curl -sSL https://mkunit.sh/install.sh | less

Arch Linux (AUR)

Available in the AUR:

# Using yay
yay -S mkunit

# Using paru
paru -S mkunit

NixOS / Nix

# Try it out
nix-shell -p mkunit

# Install permanently
nix-env -iA nixpkgs.mkunit

Or add to your configuration:

environment.systemPackages = with pkgs; [
  mkunit
];

From Source

Clone and build from source:

git clone https://github.com/alan/mkunit.git
cd mkunit
cargo build --release

# Install to /usr/local/bin
sudo cp target/release/mkunit /usr/local/bin/

Pre-built Binaries

Download pre-built binaries from the GitHub Releases page:

# Linux x86_64
curl -LO https://github.com/alan/mkunit/releases/latest/download/mkunit-linux-x86_64.tar.gz
tar xzf mkunit-linux-x86_64.tar.gz
sudo mv mkunit /usr/local/bin/

# Linux ARM64
curl -LO https://github.com/alan/mkunit/releases/latest/download/mkunit-linux-aarch64.tar.gz
tar xzf mkunit-linux-aarch64.tar.gz
sudo mv mkunit /usr/local/bin/

Verify Installation

mkunit --version

Shell Completions

Set up tab completion for your shell:

Bash

# Add to ~/.bashrc
eval "$(mkunit completions bash)"

Zsh

# Add to ~/.zshrc (before compinit)
eval "$(mkunit completions zsh)"

Fish

mkunit completions fish > ~/.config/fish/completions/mkunit.fish

See completions command for more details.

Upgrading

# Homebrew
brew upgrade mkunit

# Cargo
cargo install mkunit --force

# From source
git pull
cargo build --release
sudo cp target/release/mkunit /usr/local/bin/

Uninstalling

# Homebrew
brew uninstall mkunit

# Cargo
cargo uninstall mkunit

# Manual
sudo rm /usr/local/bin/mkunit
